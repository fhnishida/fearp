Mercados Incompletos - Parte 1¶

Considere o problema de maximizao, dados $(a_1, y_1, y_2)$: $$ W(a_1, \{y_t\}^2_{t=1}) \equiv \max_{-\bar{A}\ \le\ a_{t+1}\ \le\ y_t + a_t}\left\{ u(y_1 + a_1 - a_2) + \beta u(y_2 + a_2 - a_3) \right\} $$ em que:

  • Preferncia CRRA: $\ u(c) = \frac{c^{1-\delta}}{1 - \delta}$
  • Riquezas: $a_t$
  • Rendas: $y_t$
  • Consumos:
    • $c_1 = y_1 + a_1 - a_2$
    • $c_2 = y_2 + (1 + r)a_2 - a_3$, com:
      • $r = 0 \implies R = (1+r) = 1$
      • $a_3 = 0$ (ltimo perodo)
  • Restries, para todo $t$:
    • $a_{t+1} \ge - \bar{A}$: restrio de crdito (No-Ponzi) $\qquad$ (se $\bar{A} = 0$, no pode se endividar)
    • $c_t = y_t + a_t - a_{t+1} \ge 0$: consumo no-negativo

A resoluo do problema se d por induo retroativa:

Definio 1 (Problema perodo de vida $t=2$) $$ V_2(a_2) \equiv \max_{-\bar{A}\ \le\ a_{3}\ \le\ y_2 + a_2}\left\{ u(\underbrace{y_2 + a_2 - a_3}_{c_2}) \right\} $$

Definio 1 (Problema perodo de vida $t=1$) $$ V_2(a_2) \equiv \max_{-\bar{A}\ \le\ a_{2}\ \le\ y_1 + a_1}\left\{ u(\underbrace{y_1 + a_1 - a_2}_{c_1}) + \beta V_2(a_2) \right\} $$

Observe que, se aumentar $a_2$:

  • em $t=2$: melhora a restrio para escolha de $a_3$ e aumenta o consumo $c_2$
  • em $t=1$: diminui o consumo $c_1$, mas aumenta $V_2$

Observao 3. Note que $$ W(a_1, \{y_t\}^2_{t=1}) = V(a_1) $$

Generalizando o problema para qualquer $t$¶

In [6]:
T = 4
beta, delta = .98, 2
y_grid = np.array([1, 1, .01, .01])  # Dotaes em cada t - aqui no est generalizado, precisa preencher com a mesma quantidade de T
a_max, n_a = 2, 50
A = 0  # A-barra (p/ restrio de crdito)
a_grid = np.linspace(A, a_max, n_a)


## Problema no perodo t
V = np.zeros((n_a, T+1))  # T+1 por ltimo funo valor igual a 0
iG = np.zeros((n_a, T), dtype = np.int32)  # dtype para ter apenas nmero inteiros (ndices)

## Preencher e maximizar a funo objetivo
for t in range(T, 0, -1):
    for i_a in range(n_a):
        F_OBJ = np.zeros((n_a, n_a))
        a = a_grid[i_a]

        for i_aa in range(n_a):
            aa = a_grid[i_aa]
            c = y_grid[t-1] + a - aa

            if c < 0.:
                F_OBJ[i_a, i_aa] = -np.inf
            else:
                F_OBJ[i_a, i_aa] = u(c) + beta * V[i_aa, t]  # descontando V2

            V[i_a, t-1] = np.max(F_OBJ[i_a,:])
            iG[i_a, t-1] = np.argmax(F_OBJ[i_a,:])


# Gerando valores da funo valor a partir dos ndices de iG2
G = a_grid[iG]

ia_t = np.zeros(T+1, dtype=np.int32)
a_t = np.zeros(T+1)

for t in range(1, T+1):
    ia_t[t] = iG[ia_t[t-1], t-1]
    a_t[t] = a_grid[ia_t[t]]

    
# calculando consumo para inserir no grfico
c_t = np.zeros(T)
for i in range(T):
    c_t[i] = y_grid[i] + a_t[i] - a_t[i+1]
