Lista 3 - Exercício 1 (Modelo de Gerações Sobrepostas)¶

Considere o Modelo de Gerações Sobrepostas discutido em aula. Os indivíduos que nascem com riqueza $a_1$ e possuem trajetória de renda ao longo da vida dada por $\{y_t\}^T_{t=1}$ resolvem problema de maximização apresentado em (1)¶

\begin{align*} W\left( a_1, \{y_t\}^T_{t=1} \right) &\equiv \max_{(c_t, a_{t+1})^T_{t=1}}\left\{ \sum^T_{t=1}{\beta^{t-1} u(c_t)} \right\} \tag{1} \\ &\qquad \text{s.a. } \left\{ \begin{array}{l} c_t = y_t + (1 + r_t) a_t - a_{t+1} \ge 0 \\ a_{t+1} \ge -\bar{A} \end{array} \right. \tag{2} \end{align*}

em que $r_t$ denota a taxa de juros no período $t$, $T$ denota o horizonte de vida dos indivíduos e a utilidade instantânea é dada por¶

$$u(c) = (c^{1-\delta} - 1) / (1 - \delta).$$

Conforme discutido em aula, usando a definição da função valor $V_t(a_t)$ do período $t \in \{1, 2, ..., T\}$, a seguir: $$ V_t(a_t) \equiv \max_{-\bar{A} \le a_{t+1} \le y_t + a_t} \bigg\{ u[y_t + (1 + r_t) a_t - a_{t+1}] + \beta V_{t+1}(a_{t+1}) \bigg\} \tag{3} $$ e $V_{T+1}(a_{T+1}) = 0$, tem-se $W\left( a_1, \{y_t\}^T_{t=1} \right) = V_1(a_1)$.¶

Em cada período $s \in \mathbb{N} \equiv \{0, 1, 2, 3, ...\}$, há $T$ gerações de indivíduos presentes na economia. Há $1/T$ indivíduos nascidos no período $s$, os quais compõem a denominada geração $s$ e estão em seu primeiro período de vida ($t = 1$). Similarmente, há $1/T$ indivíduos presentes na economia em $s$, mas que nasceram no período $s-1$. Trata-se dos indivíduos da geração $s-1$, os quais estão no seu segundo período de vida ($t = 2$). Mais geralmente, para cada $t \in \{1, 2, ..., T\}$, há $1/T$ indivíduos presentes na economia no período $s$, mas que nasceram no período $s-t$. Trata-se, portanto, dos indivíduos da geração $s-t$, os quais estão no seu $t$-ésimo período de vida.¶

Supondo uma taxa de juros constante no tempo dada por $r = 0.04$ e que a renda de cada indivíduo é $y_t = 1$ se $t < 3T/4$ e $y_t = 0.1$ caso contrário, resolva os itens a seguir. Todos os programas precisam ser escritos em linguagem Python e todos os itens precisam ser resolvidos supondo $T = 10$, com a óbvia exceção do item (b).¶

import numpy as np
import matplotlib.pyplot as plt

T = 10
n_a = 50
r_ini = .04

def resolve_OLG(r=r_ini, T=T, beta=.98, delta=2, a_max=2, n_a=n_a, A=0):
    """ (r, T, beta, delta, a_max, n_a, A) -> (V, iG, G) """
    global a_grid, y_grid
    
    # Função utilidade
    def u(c):
        """ c -> utilidade """
        return ( c**(1-delta) ) / ( 1 - delta)
    
    # Criando grids necessários
    a_grid = np.linspace(A, a_max, n_a)
    y_grid = np.zeros(T)
    
    for t in range(1, T+1):  # Começa em 1 e vai até T (range desconta 1 unidade)
        if t < 3 * T / 4:
            y_grid[t-1] = 1  # índice precisa subtrair 1
        else:
            y_grid[t-1] = .1  # índice precisa subtrair 1
    
    # Objetos para preencher com as funções valor e política (índice)
    V = np.zeros((n_a, T+1))  # Cria até T+1, pois usa uma "função valor" T+1 para multiplicar por 0 na função objetivo de T
    iG = np.zeros((n_a, T), dtype=np.int32)
    
    # Preenchendo e maximizando a função objetivo
    # Começando apenas do período T (índice T-1), e vai a 0 (range desconta 1 unidade)
    # Prof. Jefferson faz os cálculos em relação ao período seguinte
    for t in range(T-1, -1, -1):  
        for i_a in range(n_a):
            F_OBJ = np.zeros((n_a, n_a))
            a = a_grid[i_a]

            for i_aa in range(n_a):
                aa = a_grid[i_aa]
                c = y_grid[t] + (1+r)*a - aa

                if c < 0:
                    F_OBJ[i_a, i_aa] = -np.inf
                else:
                    F_OBJ[i_a, i_aa] = u(c) + beta*V[i_aa, t+1]

            V[i_a, t] = np.max(F_OBJ[i_a,:])
            iG[i_a, t] = np.argmax(F_OBJ[i_a,:])
    
    V = V[:, 0:(len(V[0])-1)]  # Retirando a última coluna de zeros
    G = a_grid[iG]
    return V, iG, G

def agreg_econ(phi, r=r_ini):
    # Agregados Macroeconômicos
    C_agreg = 0
    Y_agreg = 0
    S_agreg = 0

    for t in range(T):
        for i_a in range(n_a):
            a = a_grid[i_a]
            i_aa = iG[i_a, t]
            aa = a_grid[i_aa]
            y = y_grid[t]

            c = y + (1+r)*a - aa
            medida = phi[i_a, t]  # prop. pessoas em (i_a, t)

            C_agreg += c * medida
            Y_agreg += y * medida
            S_agreg += (y - c) * medida
    
    return C_agreg, Y_agreg, S_agreg

# Realizando iterações até convergência de phi
def converg_phi(phi_ini, r=r_ini):
    norm_distr = np.inf
    tol_distr = 1e-5
    
    i = 0
    phi = phi_ini
    while norm_distr > tol_distr:
        new_phi = np.zeros((n_a, T))
        i += 1
        
        for t in range(T):
            for i_a in range(n_a):
                a = a_grid[i_a]
                i_aa = iG[i_a, t]
                aa = a_grid[i_aa]
                y = y_grid[t]
                
                c = y + (1+r)*a - aa
                medida = phi[i_a, t]
                
                # Preenchendo new_phi
                if t < T-1:
                    new_phi[i_aa, t] += medida
                else:
                    new_phi[i_aa, 0] += medida
        
        norm_distr = np.max(abs(new_phi - phi))
        phi = new_phi
    
    return phi
